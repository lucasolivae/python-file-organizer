# Organizador de Arquivos em Python

Um organizador de arquivos simples desenvolvido em Python.
Este script organiza automaticamente os arquivos de uma pasta com base em suas extensões.

## O que este script faz
- Analisa todos os arquivos da pasta atual
- Cria pastas com base no tipo de arquivo (Imagens, Vídeos, Documentos, Outros)
- Move os arquivos para as pastas apropriadas automaticamente

## Como usar

### Requisitos
- Python 3.x instalado

### Executando o script
1. Coloque o arquivo `organizer.py` dentro da pasta que deseja organizar
2. Abra um terminal nessa pasta
3. Execute o comando:

```bash
python organizer.py
